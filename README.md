# Introduction to Business Intelligence

Welcome! This repository is a comprehensive, hands-on learning resource for students beginning their journey into business intelligence and data analysis.

## 📚 About This Course

This course takes a practical, step-by-step approach to learning business intelligence fundamentals using real-world sample data and Microsoft Excel. You'll work through progressively challenging exercises that build upon each other, starting with basic data exploration and advancing to data modeling and analysis.

### What You'll Learn

- How to import and connect to data sources
- Data exploration techniques
- Data cleaning and transformation with Power Query
- Data quality assessment
- Star Schema data modeling concepts
- Relationships between tables
- Extending and evolving data models
- Visualization design principles
- How to choose the right chart for your question
- Building interactive dashboards with KPIs and slicers
- Analysis and reporting fundamentals

### Who This Course Is For

- University students studying business intelligence, data analytics, or related fields
- Beginners with little to no prior experience with data analysis
- Anyone looking to build a strong foundation in data exploration and Excel

## 📂 Repository Structure

This repository contains:

- **`data/`** - Sample CSV files representing a business scenario (orders, products, customers, vendors, etc.)

## 📊 Sample Data

The `data/` folder contains CSV files representing a fictional business:

- **orders.csv** - Order transaction data
- **products.csv** - Product catalog information
- **customers.csv** - Customer information
- **vendors.csv** - Vendor/supplier data
- **productcategories.csv** - Product category hierarchy
- **productsubcategories.csv** - Product subcategory information
- **employees.csv** - Employee information
- **vendorproduct.csv** - Product-vendor relationships

You'll start with just two files and progressively add more as you advance through the exercises.

## 💡 Learning Approach

This course follows a **learn-by-doing** methodology:

1. **Read** the exercise instructions carefully
2. **Practice** with the sample data
3. **Experiment** with different approaches
4. **Reflect** on what you've learned
5. **Ask questions** when you get stuck

Don't rush through the exercises. Take time to explore and understand each concept before moving forward.

## 🤝 Getting Help

If you encounter issues:

- Review the exercise instructions carefully
- Check that you've completed previous steps
- Verify your Excel version supports the required features
- Ensure all data files are in the correct location
